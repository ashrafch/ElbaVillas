Implementation notes

Component components/layout/WhatsAppButton.tsx — "use client" (Framer Motion entrance). Imported in app/page.tsx outside <main>, after <Footer />.
Position fixed bottom-6 right-6 z-50 — floats above all content, never overlaps the simulator modal (modal is fixed inset-0 and sits above z-50 at its own layer).
Entrance animation Framer Motion initial={{ y: 80, opacity: 0 }} → animate={{ y: 0, opacity: 1 }} with 3 s delay and custom ease [0.22, 1, 0.36, 1]. Appears after the page has loaded and the user has had time to orient.
Deep-link wa.me/{digits}?text={encoded} format. Phone stored as WA_PHONE constant — digits stripped with .replace(/\D/g, ""). Pre-filled Italian greeting message.
Icon Inline SVG WhatsApp logo, fill="#25d366" (brand green). 18 × 18 px. No external dependency.
Style #172522 bg (same brand dark as header), ring-1 ring-white/10, pill shape. Shadow deepens on hover via transition-shadow. Matches luxury tone — no loud green background.
