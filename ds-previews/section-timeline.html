Implementation notes

Component components/sections/TimelineSection.tsx — pure server component, no "use client". Imports FadeIn and cn only.
Pulse animation CSS-only .timeline-pulse class defined in globals.css (@keyframes timeline-pulse). No Framer Motion needed — safe for server render.
Status system Type "done" | "current" | "upcoming". Dot sub-component renders the appropriate dot shape. Opacity/color classes cascade from status.
Section background #efe7d8 (sabbia) — warm editorial break between dark InvestmentSection and lighter ContactLeadSection. Matches the marquee strip color.
Entrance <FadeIn> wraps heading block (delay 0) and timeline body (delay 0.12s). Standard pattern used across all sections.
Page position Inserted between <InvestmentSection /> and <ContactLeadSection /> in app/page.tsx. Anchored id="avanzamento" for nav linking.
