Animation sequence (v2 — client component)

Trigger useInView(ref, { once: true, margin: "-80px" }) — all animations fire once on scroll, no replay.
Base line scaleX: 0 → 1 with transformOrigin: "left", 0.9 s. Gives the "drawing" effect.
Progress fill Separate div inside overflow: hidden — width: 0% → 50%, delay 0.12 s. 50% = index 2 of 4 intervals = current milestone.
Dots Staggered spring pop-in (stiffness: 320, damping: 22). Done/current use spring; upcoming use ease.
Current dot Ring pulses with Framer Motion keyframes [1, 2.4, 1] scale + [0.55, 0, 0.55] opacity, infinite. "in corso" badge breathes opacity independently.
Mobile line Vertical line uses scaleY: 0 → 1, transformOrigin: "top". Rows slide in from x: -14 with stagger 0.12 s/item.
